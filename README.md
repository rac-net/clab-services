RampUp to Automation part6 - Ansible Services

This is a repository containing all the files used to create the segment-routing network using ansible and l3vpn and vpls services

it includes the Containerlab script, plus the ansible playbooks for both Nokia and Cisco using netconf for both services

The playbook creates a segment-routing network from scratch.


This scripts suposes that you create the containerlab deployment
